# Step Diagram Images

Drop AI-generated diagram images here using the filename `<step-id>.<ext>`.

Supported extensions: `.png`, `.jpg`, `.jpeg`, `.webp`

The app auto-discovers these via Vite glob — no code changes needed when you add a new image, just refresh the dev server.

See `docs/step-image-prompts.md` for the full prompt list and the list of step IDs.
